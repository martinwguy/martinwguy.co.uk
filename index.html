Hi!

I've been programming since 1977 and studied and then lectured in computing and did Unix support 1982-1990 at University of Kent at Canterbury, IRCAM and others. You'll find some stuff from the eighties on the famous tape.

In the 90s I helped set up and run the FreakNet MediaLab in Catania, Sicily and then Lumpy Street in Canterbury, UK and in 2006 bootstrapped a new port of the Debian operating system.

I publish and/or maintain the following sites:

  • Early synthetic computer music made using CSound
  • A collection of digital books
  • Una collezione di libri digitali
  • Delia Derbyshire's audiological chronology
  • recwav: Fast audio spectrograms from slow hardware
  • xvi: The smallest full-function vi clone
  • bigfloat: An infinite-precision math library for Miranda and Haskell
  • Fixing GCC's code generation for the MaverickCrunch FPU
  • Better code generation for AVR32 processors
and recent projects include
  • The software and documentation for the Sim.One computer
    • support for the board in the mainline Linux kernel
    • fixes and workarounds for the chip's hardware bugs
    • a new port of Debian for its operating system
    • a fixed GCC to enable use of the board's MaverickCrunch floating point unit
    • repositories of accelerated Debian packages using the FPU, which run 4 times faster
  • The software and documentation for the Mizar32 computer including
    • completion of the AVR32UC3A chip support in eLua to create the board's firmware
    • code size optimizations for the 128Kb flash version of the firmware
    • an optimized I2C driver in PIC32 assembler, dozens of times faster
    • its User Guide
