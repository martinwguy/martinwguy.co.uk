Hi!

I've been programming since 1977 and studied and then lectured in computing and did Unix support 1982-1990 at University of Kent at Canterbury, IRCAM and others. You'll find some stuff from the eighties on the famous tape.

In the 90s I helped set up and run the FreakNet MediaLab in Catania, Sicily and then Lumpy Street in Canterbury, UK and in 2006 bootstrapped a new port of the Debian operating system.

I publish and/or maintain the following sites:

  • Early synthetic computer music made using CSound
  • A collection of digital books
  • Una collezione di libri digitali
  • Delia Derbyshire's audiological chronology and the WikiDelia
  • recwav: Fast audio spectrograms from slow hardware
  • xvi: The smallest full-function vi clone
  • bigfloat: An infinite-precision math library for Miranda and Haskell
  • Fixing GCC's code generation for the MaverickCrunch FPU and repositories of accelerated Debian packages compiled to use it (average: 2.5 to 4 times faster)
  • A GCC cross-compiling system giving better code generation for AVR32 processors
and recent projects include
  • The software and documentation for the Sim.One computer adding support for it to the mainline Linux kernel and competing/fixing its device drivers
  • Creating a new port of Debian for ARM processors
  • Completion of the AVR32UC3A system-on-chip support in embedded Lua: writing the missing hardware-driving modules and fixing bugs in existing ones
  • code size optimizations to fit as much of embedded Lua as possible into 120Kb of Flash memory
  • an optimized I2C driver in PIC32 assembler, dozens of times faster
  • Writing and typesetting the Mizar32 Quick Start Guide, a tutorial and programming guide for a tiny 20-euro hobbyist computer.
